# 🔨 Whack-a-mole (두더지 잡기)

1인용 (Human vs System) 두더지 잡기 게임.

## 📦 기본 정보

* **저장소:** `~/github/Whack-a-mole`
* **스택:** TypeScript + HTML5 Canvas + IndexedDB (외부 라이브러리·프레임워크 0%)

---

## 🎮 핵심 기능

### 1. 100% Full HTML5 Canvas 렌더링
* **순수 Canvas UI:** DOM 엘리먼트 없이 게임 메뉴, 버튼, 스코어보드, 텍스트, 별점 UI 등 모든 인터페이스를 Canvas 내에서 직접 드로잉.
* **통합 렌더 루프:** `requestAnimationFrame`을 사용하여 상태 업데이트와 화면 갱신을 동기화.

### 2. 게임 플레이 및 스테이지/별점 시스템
* **스테이지 승급:** 매 스테이지마다 '목표 점수(Goal Score)'를 설정. 제한 시간 내 목표 점수 도달 시 다음 스테이지로 자동 전환.
* **별점 시스템(Star Rating):** 스테이지 완료 시 성과(잔여 시간, 정확도, 콤보 횟수)를 종합 산출하여 Canvas 상에 별 애니메이션과 함께 1~3개 부여.
* **다양한 타겟 타입 (스테이지 연동형):**
  * **일반/황금 두더지:** 타격 시 점수 및 콤보 획득.
  * **폭탄 두더지:** 타격 시 감점 또는 라이프 차감.
  * **🌸 [New] 10스테이지 이후 트랩 (꽃):** **10판부터는 두더지 대신 일정 확률로 '꽃'이 출현. 꽃을 때리면 안 되며, 타격 시 콤보 리셋 및 치명적인 페널티(점수 대폭 차감 또는 라이프 차감) 부여.**
* **콤보 및 피버 시스템:** 연속 타격 성공 시 점수 배율 증가 및 피버 타임(화면 전체 파티클 이펙트).

### 3. 난이도 조절 및 상태 관리
* **출현 패턴 휴리스틱:** 스테이지가 올라갈수록 두더지 등장 속도, 출현 간격, 특수 객체(폭탄/꽃) 확률을 정밀하게 제어하는 상태 머신 엔진. 10스테이지 이후부터는 피지컬과 판단력을 동시에 요구하는 복합 패턴 생성.
* **IndexedDB 자동 저장:** 최고 기록, 스테이지 진행 상태, 각 스테이지별 획득 별점 정보를 매 판 종료 후 자동 저장.

### 4. 반응형 스케일링 및 입력
* **디스플레이 최적화:** 와이드/세로형(폴더블폰) 화면에 맞춰 그리드 배치를 자동 최적화하고, 캔버스 비율을 동적으로 스케일링.
* **멀티 입력:** 터치/마우스 클릭 좌표 변환을 통한 정밀 타격 판정 및 키보드(텐키패드/QWE-ASD-ZXC) 매핑 지원.

### 5. 미디어 및 리소스 (라이선스 프리)
* **그래픽:** CC0 무료 이미지 활용 및 **Canvas API 절차적 생성(Procedural Generation)**으로 두더지 및 꽃(arc와 bezierCurve 이용) 벡터 드로잉 병행.
* **오디오:** 저작권 없는 효과음 활용 및 Web Audio API `OscillatorNode`를 이용한 신시사이저 방식의 효과음(타격음, 꽃 타격 시 삐- 하는 경고음) 직접 생성.

### 6. 안정성
* **에러 핸들링:** 애니메이션 예외 처리, IndexedDB 연결 유실 시 LocalStorage 백업, 미디어 로딩 실패 시 Canvas 폴백(Fallback) 도형 렌더링.
